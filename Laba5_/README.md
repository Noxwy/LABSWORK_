# Пример реализации паттернов Абстрактная фабрика и Строитель<br>
 В данной лабораторной работе, программа имеет фабрики с продуктами(в данном случае магазины Ашан и Азбука Вкуса). <br>
 Из купленных продуктов, повар в ресторане, может приготовить различные блюда из этих продуктов, используя при этом различные<br>
 способы приготовления.
